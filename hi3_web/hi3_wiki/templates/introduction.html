INTRODUCTION TO THE GAME

Honkai Impact 3 is a action RPG taking many insperations from action game like Devil May Cry series, Bayonetta and Metal Gear Rising: Revengeance, etc. It has 2 types of main contents, story mode and end game challenges. Story mode is a long, complex story, mainly as the form of a visual novel game while end game challenges are where the most combat happened. Aside from that, there are also open-world mode and various events

We will not talk about end game challenges in this because all of them have dedicated guides. There are 3 main end game challenges: Elysian Realm, Abyss, Memorial Arena

PLAYABLE CHARACTERS

This is the first confusing part of this game.

Character Stats

  • HP - How much damage a Character can take before falling in combat
  • ATK - How much damage a Character deals
  • DEF - Reduces the damage a Character takes
  • Speed - Determines how fast and how often a Character will act in combat
  • Crit Rate - How likely it is for the Character to land a critical hit
  • Crit DMG - Determines the damage multiplier when landing a critical hit
